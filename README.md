# Hướng dẫn chạy chương trình:

- b3: npm start 
   - note: 
      - frontend localhost: 3000
      - backend localhost: 4000
- note: để chạy riêng client: tại ./frontend : npm start
- note: để chạy riêng server: tại ./backend : npm start
